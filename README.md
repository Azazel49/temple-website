# Indian Temple Website

A professional, full-stack website for Indian temples with integrated payment gateways for donations and event bookings.

## Features

### Current (Frontend Demo)
- 🏛️ Temple Information & History
- 🙏 Darshan Scheduling System
- 📅 Event/Puja Booking
- 💝 Donations & Offerings System
- 📧 Newsletter Subscription
- 📱 Fully Responsive Design
- 🔒 Security Headers & Cloudflare Ready
- ⚡ Performance Optimized
- 🎨 Modern UI/UX

### Upcoming (Backend)
- 💳 Payment Gateway Integration (Razorpay, PhonePe/Google Pay)
- 👤 User Authentication & Accounts
- 📊 Admin Dashboard
- 📧 Email Notifications
- 📱 Mobile App Integration

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Azazel49/temple-website.git
cd temple-website
```

2. Open `index.html` in your browser or use a local server:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (using http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser
